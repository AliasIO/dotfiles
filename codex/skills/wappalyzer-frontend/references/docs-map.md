## Working Rule

- Re-open the official page whenever a change depends on exact prop names, slot names, helper-class syntax, accessibility behavior, or responsive behavior.
- Use the durable notes in this skill to remember the documented defaults; do not substitute them for the docs when a task is component-specific.
